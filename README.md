# vue-bot

> Bot Build With Vue

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

Supported Inputs
-----------------
Text
Button Group[single]
Card Group[single]

Support Output
----------------
Text
Image
